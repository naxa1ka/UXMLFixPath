## Install

1. Open the Package Manager from `Window > Package Manager`
2. `"+" button > Add package from git URL`
3. Enter the following
   * https://github.com/naxa1ka/UXMLSrcFixer.git?path=Assets/Plugins/Nxlk/UXMLSrcFixer#tag
